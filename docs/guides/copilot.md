# Co-pilot / Assistant

Co-pilot -- встроенный AI-ассистент Pulsar AI, работающий в двух режимах: командный режим
(доступен всегда) и LLM-режим (требует OpenAI API ключ). Помогает управлять обучением,
анализировать данные и получать рекомендации через естественный язык.

---

## Два режима работы

| Режим      | Требования             | Возможности                              |
|------------|------------------------|------------------------------------------|
| Command    | Нет                    | Slash-команды, быстрые действия          |
| LLM        | `OPENAI_API_KEY`       | Естественный язык, 14 инструментов       |

!!! info "Активация LLM-режима"
    ```bash
    export OPENAI_API_KEY=sk-your-key-here
    ```
    При наличии ключа LLM-режим включается автоматически. Используется модель
    `gpt-4o-mini` для баланса между скоростью и качеством.

---

## Командный режим: Slash-команды

Slash-команды доступны в чате Co-pilot без необходимости в API-ключе:

| Команда         | Описание                                          |
|-----------------|---------------------------------------------------|
| `/status`       | Текущий статус системы и активных задач            |
| `/datasets`     | Список загруженных датасетов                       |
| `/train`        | Запуск обучения с указанным конфигом               |
| `/recommend`    | Рекомендации параметров для модели и данных         |
| `/hardware`     | Информация о доступном оборудовании                |
| `/experiments`  | Список и статус экспериментов                      |
| `/workflows`    | Список доступных workflow                          |
| `/estimate`     | Оценка времени и стоимости обучения                |
| `/cancel`       | Отмена текущего обучения                           |
| `/preview`      | Предпросмотр датасета                              |
| `/help`         | Справка по командам                                |

### Примеры использования

```
/status
> Активных задач: 1
> Обучение: sft-llama3-v2 (эпоха 2/3, loss: 0.84)
> GPU 0: 87% utilization, 18.2/24.0 GB VRAM
```

```
/recommend meta-llama/Llama-3-8B --dataset my-chat-data --task sft
> Рекомендуемые параметры для SFT Llama-3-8B:
>   learning_rate: 2e-4
>   batch_size: 4 (gradient_accumulation: 4)
>   lora_r: 16, lora_alpha: 32
>   epochs: 3
```

```
/estimate configs/sft_config.yaml
> Оценка обучения:
>   Время: ~2ч 15мин (3 эпохи, 10k samples)
>   VRAM: ~18 GB (LoRA rank 16)
>   Рекомендуемый GPU: RTX 4090 / A100
```

## LLM-режим

При наличии `OPENAI_API_KEY` Co-pilot использует GPT-4o-mini с 14 специализированными
инструментами для работы с Pulsar AI.

### Список инструментов

| Инструмент               | Описание                                    |
|--------------------------|---------------------------------------------|
| `list_experiments`       | Получить список всех экспериментов           |
| `get_experiment`         | Детали конкретного эксперимента              |
| `start_training`         | Запустить обучение с конфигурацией            |
| `check_training`         | Проверить статус текущего обучения            |
| `cancel_training`        | Отменить обучение                            |
| `list_datasets`          | Список доступных датасетов                   |
| `preview_dataset`        | Предпросмотр записей датасета                |
| `recommend_params`       | Рекомендации гиперпараметров                 |
| `get_hardware`           | Информация о GPU/CPU                         |
| `run_evaluation`         | Запустить оценку модели                      |
| `list_workflows`         | Список workflow                              |
| `get_workflow`           | Детали конкретного workflow                  |
| `estimate_training_cost` | Оценка времени и стоимости                   |
| `suggest_config`         | Сгенерировать конфигурацию обучения          |

## Примеры на естественном языке

LLM-режим позволяет общаться с Co-pilot свободным текстом:

!!! tip "Примеры запросов"

    **Анализ данных:**
    ```
    Покажи первые 5 записей из датасета alpaca-cleaned
    ```

    **Рекомендации:**
    ```
    Какие параметры посоветуешь для файнтюнинга Llama 3 8B на 50k примерах?
    ```

    **Управление обучением:**
    ```
    Запусти SFT обучение с конфигом configs/sft_config.yaml
    ```

    **Диагностика:**
    ```
    Почему последний эксперимент завершился с ошибкой?
    ```

    **Планирование:**
    ```
    Сколько времени займёт обучение Mistral 7B на моём GPU?
    ```

    **Сравнение:**
    ```
    Сравни результаты последних трёх экспериментов
    ```

### Пример диалога

```
Пользователь: Какой из моих экспериментов показал лучший eval_loss?

Co-pilot: Я проверил ваши эксперименты. Лучший eval_loss у эксперимента
exp_20260302_091015 (sft-llama3-v2): 0.58.

Вот топ-3:
1. sft-llama3-v2  — eval_loss: 0.58
2. sft-llama3-v3  — eval_loss: 0.61
3. sft-llama3-v1  — eval_loss: 0.69

Хотите использовать параметры лучшего эксперимента для нового обучения?
```

!!! warning "Лимиты LLM-режима"
    Co-pilot использует GPT-4o-mini через OpenAI API. Каждый запрос расходует токены.
    Стоимость одного запроса обычно составляет менее $0.01, но при интенсивном
    использовании рекомендуется отслеживать расход через OpenAI Dashboard.
